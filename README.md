# Suivi des Marées

Application de gestion maritime pour le suivi des marées.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build Production

```bash
npm run build
```

## Déploiement Vercel

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically

## Structure des fichiers CSV

### proprietaires.csv

```
Matricule,Barque,Propriétaire
```
